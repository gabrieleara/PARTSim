◆ changeBudget()

Tick RTSim::PollingServer::changeBudget ( const Tick &  new_budget)
virtual

Dynamically changes the budget.

Parameters
new_budgetthe new budget. Can be higher or lower than the current one
Returns
the function returns the time at which the new budget has been imposed. Usually, an increase in the budget is immediately enforced, so it returns the current time. A decrease in the budget, instead, cannot be enforced immediately, so the function will return the time in the future when the new budget has been enforced.

Implements RTSim::Server.
