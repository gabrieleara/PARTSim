Detailed Description

Contains the scheduling parameters and a pointer to the task. It is used by a scheduler to store the pointer to the task and the set of scheduling parameters.

Each scheduler has its own task model. So the class inheritance trees of the task models and of the schedulers are similar.

◆ raiseThreshold()

void RTSim::TaskModel::raiseThreshold ( )

This function raises the threshold of the current task. This is equivalent to :

  • saving the old value of the priority
  • setting the new priority value to th.

You must make sure that the task model on which this function is called MUST BE the highest priority task.

◆ restorePriority()

void RTSim::TaskModel::restorePriority ( )

Restores the original task priority. This is the opposite of raiseThreshold, and must be performed on the currently executing task, after it has been removed from the queue. In some cases, it may cause a change of context.
